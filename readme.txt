Set minimum and maximum purchase quantities for selected WooCommerce products and product categories.

== Description ==

Quantity Ceiling adds a WooCommerce settings page for assigning minimum and maximum purchase quantity rules to specific products or product categories.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/quantity-ceiling` directory, or install the plugin zip from the WordPress Plugins screen.
2. Activate the plugin through the Plugins screen in WordPress.
3. Go to Products > Quantity Ceiling to configure quantity rules.
