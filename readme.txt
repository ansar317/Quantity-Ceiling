Control purchase quantities for WooCommerce products and categories. Prevent underselling and enforce bulk order limits effortlessly.

== Description ==

The **Quantity Ceiling** plugin provides a streamlined solution for WooCommerce store owners to enforce strict purchasing boundaries. Whether you need to prevent small, unprofitable orders or manage inventory by limiting bulk purchases, this plugin gives you total control.

### Key Features
*   **Product-Level Control:** Set specific minimum and maximum quantity limits for individual products.
*   **Category-Level Rules:** Apply quantity restrictions to entire product categories to save time.
*   **Seamless WooCommerce Integration:** Native settings page located under the 'Products' menu.
*   **User-Friendly Error Handling:** Automatically notifies customers if their cart quantities do not meet your specified rules.
*   **Lightweight & Efficient:** Developed with clean, optimized code for high performance.

== Installation ==

1.  **Upload:** Download the plugin ZIP file and upload it to the `/wp-content/plugins/` directory via FTP, or install it directly via the WordPress Admin dashboard (Plugins > Add New > Upload Plugin).
2.  **Activate:** Navigate to the **Plugins** screen and click **Activate** for Quantity Ceiling.
3.  **Configure:** Go to **Quantity Ceiling** in your WordPress admin sidebar (its own top-level menu) to set your minimum and maximum quantity rules.

== Frequently Asked Questions ==

= Does this work with variable products? =
Yes, Quantity Ceiling is designed to integrate with standard WooCommerce product types.

= Can I override rules for specific products? =
Yes, you can set granular rules per product that override general category settings.

= Where do I set the rules? =
Once activated, you will find the configuration panel under its own **Quantity Ceiling** menu in your WordPress admin sidebar.
